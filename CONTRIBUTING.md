# 贡献指南

欢迎提交院校项目数据、匿名案例、界面改进和推荐规则修正。

## 数据 PR

1. 只提交有权公开的数据；
2. 优先提供学校或学院官方来源；
3. 每条记录填写 `source_url`、`data_year` 和 `evidence_level`；
4. 不包含个人可识别信息；
5. 运行 `python scripts/validate_programs.py <文件>`；
6. 在 PR 中解释新增或修改字段的证据。

## 代码 PR

- 新逻辑必须有测试；
- 不得把匹配分描述为录取概率；
- 涉及权重调整时说明依据及对示例的影响；
- 保持前端对移动设备可用。
